# PadalaLock
Controlled remittance for OFWs

## Problem
Money sent home is sometimes misused

## Solution
Lock funds and release only to intended recipients

## Timeline
2–3 days MVP

## Stellar Features
USDC, Soroban

## Vision
Smart remittance with accountability

## Build
soroban contract build

## Test
cargo test

## Deploy
soroban contract deploy

## Example
soroban contract invoke --id <ID> --fn deposit --arg user --arg amount
